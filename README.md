# CNN-Tutorial

Overview:
This repository contains a tutorial exploring how Convolutional Neural Networks (CNNs) work internally by visualising filters and feature maps.

What it does:
Loads an example image
Passes it through a CNN
Visualises:
Learned filters
Feature maps at different layers

This helps show how CNNs detect edges, patterns, and increasingly complex features.

1. Open the notebook: CNN_Tutorial.ipynb
2. Run all cells to see the visualisations.

Notes:
You can replace the input image with your own, if not it uses an image library so no dataset required
Can be used offline
No training required (uses pretrained model)

This tutorial was created as part of a machine learning coursework assignment, with the goal of making CNNs more interpretable and accessible.
